# Grunt 前端自动化模板
## 步骤
	- 进入项目目录 npm install -save-dev 安装npm所需模块
	- grunt serve 开启服务
	- 对应自己的项目 修改Gruntfile.js 
## 项目目录
    .
    ├── dist // 生产环境
    └──  src // 源代码
    	   ├── assets // JS/CSS/IMG
    	   └── page //  页面
